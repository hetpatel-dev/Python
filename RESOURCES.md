# Python & DSA Resources

## Knowledge

- [Python Official Tutorial: An Informal Introduction](https://docs.python.org/3/tutorial/introduction.html)
  Covers basic types, variables, strings, lists. Use for: Lesson 1 fundamentals.

- [Python Data Model: Objects, values and types](https://docs.python.org/3/reference/datamodel.html#objects-values-and-types)
  Official spec on object identity, type, mutability. Use for: understanding what variables really are.

- [Python Simple Statements: Assignment](https://docs.python.org/3/reference/simple_stmts.html#assignment-statements)
  Official spec on assignment, augmented assignment, annotated assignment, del. Use for: variable mastery.

- [Python Execution Model: Naming and binding](https://docs.python.org/3/reference/executionmodel.html#naming-and-binding)
  Official spec on scopes, name resolution, global/nonlocal. Use for: understanding variable scope.

- [PEP 8 — Style Guide for Python Code](https://peps.python.org/pep-0008/)
  Naming conventions and code style. Use for: writing idiomatic Python.

- [PEP 526 — Syntax for Variable Annotations](https://peps.python.org/pep-0526/)
  Type hint syntax for variables. Use for: type annotations.

## Wisdom (Communities)

- [r/learnpython](https://reddit.com/r/learnpython)
  Beginner-friendly Python community. Good for: getting unstuck on basic concepts.
