Lesson 1: Python Basics — Variables & Output

concept Python Basics

Welcome! The goal of this lesson is simple: write a Python program that asks for someone's name, stores it, and prints a personalized greeting. That's your first tangible win.

What is Python?

Python is a programming language designed for readability. You write instructions in plain-English-like syntax, and Python executes them line by line.

You have Python installed (v3.14). There are two ways to run Python:

  • REPL — type python in your terminal and get an interactive prompt. Good for experimenting.
  • Script — write code in a .py file and run it with python filename.py. That's what we'll do.

Your first program

Create a file called hello.py and write:

print("Hello, world!")

Run it:

python hello.py

You'll see Hello, world! printed. The print() function outputs text to the console.

Variables

A variable is a named box that holds a value. You create one with =:

name = "Alice"
age = 25
height = 1.68
is_student = True

Python figures out the type automatically. You don't need to declare it.

TypeNameExample
intInteger42, -3, 0
floatDecimal number3.14, -0.5
strString (text)"hello", 'hi'
boolBoolean (True/False)True, False

Use type() to check:

print(type(42))       # <class 'int'>
print(type("hello"))  # <class 'str'>

Comments

Anything after # is ignored. Use them for notes.

# This is a comment — Python won't run it
print("This will run")  # inline comment

Getting user input

input() reads a line from the user. It always returns a string.

name = input("What is your name? ")
print("Hello, " + name)

f-strings (the best way to format text)

Put an f before the string and use {} to insert variables:

name = "Alice"
age = 25
print(f"{name} is {age} years old.")

This is cleaner than using + to concatenate. Get used to f-strings early — you'll use them constantly.

Your mission win

Write a script that:

  1. Asks for the user's name
  2. Asks for their age
  3. Prints: "Hello [name]! You are [age] years old."

Try it now. Create a greeting.py file, write the code, and run it. This is your first tangible win.

Solution (check after you try)

Reveal solution 
name = input("What is your name? ")
age = input("How old are you? ")
print(f"Hello {name}! You are {age} years old.")

Practice exercises

Create each exercise as a separate .py file in exercises/01-python-basics/. Run each one to verify.

Exercise 1: About Me

Create three variables: city, country, population. Assign values and print a sentence like: "I live in [city], [country]. Population: [population].".

Exercise 2: Simple Calculator

Ask the user for two numbers (input() returns strings — you'll need int() to convert). Print their sum, difference, product, and quotient.

Hint: int("5") converts the string "5" to the integer 5.

Exercise 3: Type Explorer

Create a variable of each type (int, float, str, bool). Print each value and its type using type().

Output should look like:

42 is <class 'int'>
3.14 is <class 'float'>
hello is <class 'str'>
True is <class 'bool'>

Exercise 4: Age in Days

Ask for the user's age in years. Convert to an integer, calculate their approximate age in days (age * 365), and print: "You are approximately [days] days old.".

Exercise 5: Mad Libs

Ask the user for: a noun, an adjective, and a verb. Then print a funny sentence using all three. For example: "The [adjective] [noun] decided to [verb]!".
